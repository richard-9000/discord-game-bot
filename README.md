# discord-game-bot
A very simple bot to control games from a discord channel

## Quick and simple install:

1. Run

```git clone https://github.com/richard-9000/discord-game-bot
cd discord-game-bot
python3 -m venv .
```

2. Edit the poker.ini to add your API key

3. Run the poker.py

```python3 poker.py
```
